# Alpha Signal Library

## Overview

This repository contains signal-level implementations derived from research.

Each signal is designed to capture a specific market behavior.

---

## Signals

### 1. Moving Average Slope

**Idea**
Momentum is better captured by the slope of a moving average than by price position alone.

**Logic**

* Calculate moving average
* Measure rate of change
* Apply threshold filter

**Usage**

* Trend filter
* Position sizing
* Entry condition

---

## Philosophy

Signals are not strategies.

They are building blocks for decision systems.
